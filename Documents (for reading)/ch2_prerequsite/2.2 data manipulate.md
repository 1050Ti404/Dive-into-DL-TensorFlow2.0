## 2.2.5 saving memory


```python
X = tf.Variable(X)
Y = tf.cast(Y, dtype=tf.float32)

before = id(Y)
Y = Y + X
id(Y) == before
```




    False




```python
Z = tf.Variable(tf.zeros_like(Y))
before = id(Z)
Z[:].assign(X + Y)
id(Z) == before
```




    True




```python
Z = tf.add(X, Y)
id(Z) == before
```




    False




```python
before = id(X)
X.assign_add(Y)
id(X) == before
```




    True
